Join meeting in browser not app. Open dev tools > console > type 'allow pasting' if nessecary > paste the script. Autosaves every 5 minutes. 

 ```TeamsCaps.text()```   -> transcript as a string
 
 ```TeamsCaps.save()```   -> download .txt + .json
 
 ```TeamsCaps.copy()```   -> copy to clipboard
 
 ```TeamsCaps.stop()```   -> stop polling
 
 ```TeamsCaps.debug()```  -> dump caption DOM (if capture looks broken)
